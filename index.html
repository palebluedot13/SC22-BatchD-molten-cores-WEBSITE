Suicide Attempt Prediction

Using existing data to predict if someone may by at risk of attempting suicide.

Get Started

About the Suicide Prediction Index

The Suicide Prediction Index uses data from the ForeverAlone dataset to effectively predict suicidal behaviors and thoughts based on gender, age, race, etc. With this product, you are able to easily learn the key factors that contribute to suicide risk. Having this information on hand can help you recognize the signs of someone who is at risk of attempting suicide so that you can work to help prevent it before it becomes serious.

Our Project

Random Forest

The random forest is a classification algorithm consisting of many decisions trees. It uses bagging and feature randomness when building each individual tree to try to create an uncorrelated forest of trees whose prediction in a group is more accurate than that of any individual tree.

Random Forest Classifier

This model had one of the higher accuracies out of all the models. It had a relatively low number of false positives and negatives, it was good at predicting true negatives, and it was okay at predicting true positives overall, but compared to the other models it was still really good in each category.
The true negative had a precision of 0.8, recall of 0.83, and an F1 score of 0.81.
The true positive had a precision of 0.43, recall of 0.39, and F1 score of 0.41.
The model did very well, although not perfectly.

Support Vector Classifier

Classifying data is an essential part of machine learning. SVC is a powerful supervised machine learning method used for category detection. It divides the data set into classes for predictions. The module used by scikit-learn is sklearn.svm.SVC.

SVC

This model had the least accuracy of 0.49, though it was able to predict the true positives accurately. The confusion matrix, on the right, is used to summarize the performance of the classification algorithm. It had f1 scores of 0.58 and 0.36 which is the harmonic mean of the precision and accuracy together.

XGBoost

XGBoost provides parallel tree boosting and is the leading machine learning library for regression, classification, and ranking problems.

XGBoost

Here, you can see that XGBOOST is not the best, as false Negatives are higher than actual true positivies. Additionally, F1-score is the harmonic mean of precision and recall .So, F1 score for people who are going to do suicide is 0.21 and who are not going to do suicide is 0.81.

KNN

The K-Nearest Neighbors algorithm predicts the values of new datapoints based on how closely it matches the other points in the training set. This specific KNN model estimates how likely a person is to commit suicide based on the seven nearest data points.

KNN

The KNN model had an accuracy of 0.59 for the f1-score which is the percent of positive predictions that were correct. A f1-score of 1.0 is the most accurate and a score of 0.0 is the least. The confusion matrix shows that the KNN model has a lot of true negatives but not a lot of true positives.

Logistic Regression

Logistic Regression is an important part of our data used to estimate the possibility. With this we were able to predict an individuals attempt to suicide. And then we can send immeadiat help to those at high risk of suicide. Therefor saving many lives.

Logistic Regression

The accuracy of this model showed that 15 people wanted to attempt suicide however 8 people after the surveyed committed suicide. With this data we were accurately able to predict with 92% accuracy. With this we can immeadiately find those at risk to suicide and stop them from committing suicide.
